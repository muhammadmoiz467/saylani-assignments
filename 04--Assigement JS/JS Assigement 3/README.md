# ⚡ JavaScript Assignment 3 - Conditions & Prompts

## 📌 Project Description

Yeh ek interactive web project hai jo **HTML, CSS, Bootstrap aur JavaScript** use karke banaya gaya hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke JavaScript assignment ka hissa hai.

Is project me user prompts aur conditions ke zariye different operations perform karta hai aur result screen par display hota hai.
