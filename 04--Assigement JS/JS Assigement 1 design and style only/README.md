# ⚙️ JavaScript Assignment 1 - UI Design

## 📌 Project Description

Yeh ek basic user interface (UI) project hai jo **HTML, CSS aur Bootstrap** use karke banaya gaya hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke JavaScript assignment ka hissa hai.

⚠️ Note: Is project me abhi sirf **design/UI banayi gayi hai**, JavaScript functionality (logic) abhi implement nahi ki gayi.

## ✨ Features

* Clean UI layout for JavaScript practice
* Buttons for different JS tasks (Alerts, Variables, Math operations)
* Input fields for statements and output
* Bootstrap-based responsive layout
* Navigation bar with search

## 📄 Sections Included

* Header / Navbar

* Footer
