# ⚡ JavaScript Assignment 4 - Arrays & Loops

## 📌 Project Description

Yeh ek interactive web project hai jo **HTML, CSS, Bootstrap aur JavaScript** use karke banaya gaya hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke JavaScript assignment ka hissa hai.

Is project me user input, arrays aur loops ka use karke different operations perform karta hai aur result dynamically screen par display hota hai.
