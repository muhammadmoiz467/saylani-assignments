This assignment is part of **Saylani Mass IT Training (SMIT)** JavaScript assignments.

* 🌐 Get Current URL
* 🔗 Redirect to another website
* 🔄 Replace current URL
* 🪟 Open Popup Window
* ✏️ Add content inside Popup
* 🌍 Assign URL to Popup
* ❌ Close Popup Window
