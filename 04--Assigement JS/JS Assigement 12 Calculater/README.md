## 📚 Assignment Details

This assignment is part of **Saylani Mass IT Training (SMIT)** JavaScript assignments.

* 🔢 Number Input (0–9, 00, .)
* ➕ Basic Arithmetic Operations (+, −, ×, ÷)
* 🧮 Calculate Result (=)
* ❌ Clear Display (C)
* ⚡ Real-time Display Update
