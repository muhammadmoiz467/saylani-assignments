=# ⚡ JavaScript Assignment 2 - Interactive UI

## 📌 Project Description

Yeh ek interactive web project hai jo **HTML, CSS, Bootstrap aur JavaScript** use karke banaya gaya hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke JavaScript assignment ka hissa hai.

Is project me user buttons par click karke different JavaScript operations perform kar sakta hai aur result screen par display hota hai.

## 🏫 Program

## 📚 Chapters Covered

* Chapter 1: Alerts
* Chapter 2: Variables for Strings
* Chapter 3: Variables for Numbers
* Chapter 4: Variable Names (Legal & Illegal)
* Chapter 5: Math Expressions
* Chapter 6: Increment & Decrement
* Chapter 7: Math Expressions (Advanced)
