# 📱 CSS Assignment 2 - iPhone Style Website

## 📌 Project Description

Yeh ek modern layout website hai jo **HTML aur CSS** use karke banayi gayi hai. Yeh project **Saylani Mass IT Training Program** ke assignment ka hissa hai, jiska purpose layout design, typography aur section-based structure ko practice karna hai.

## 🚀 Live Preview

👉 https://moiz-css2.surge.sh

## 🏫 Program

* Saylani Mass IT Training Program (SMIT)

## ✨ Features

* Clean and modern UI design
* Google Fonts integration (Raleway)
* Hero section with call-to-action
* Image and text based layout
* Informational sections with statistics
* Card-style components
* Footer section

## 📄 Sections Included

* Header Section (Intro + Button)
* Hero Section (Image + Stats)
* Mission Section
* Images Section
* Cards Section
* Footer

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
