# 🎨 CSS Time Table Project

## 📌 Project Description

Yeh ek **Time Table website** hai jo **HTML aur CSS** use karke banayi gayi hai. Is project ka purpose layout design, table styling aur light/dark mode UI ko practice karna hai.

## 🚀 Live Preview

👉 https://moiz-css1.surge.sh

## ✨ Features

* Responsive Time Table layout
* Clean and structured table design
* Light Mode & Dark Mode toggle
* Styled using pure CSS
* Beginner-friendly project

## 📄 Pages Included

* Light Mode Page (`index.html`)
* Dark Mode Page (`dark.html`)

## 🛠️ Technologies Used

* HTML5
* CSS3
