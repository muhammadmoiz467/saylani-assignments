# 🏫 CSS Assignment 3 - Saylani Website Clone

## 📌 Project Description

Yeh ek modern multi-section website hai jo **HTML aur CSS** use karke banayi gayi hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke assignment ka hissa hai, jisme real-world website layout ko design aur structure karna seekha jata hai.

## 🚀 Live Preview

👉 https://moiz-css3.surge.sh

## 🏫 Program

* Saylani Mass IT Training Program (SMIT)

## ✨ Features

* Multi-section modern website layout
* Navigation bar with links and buttons
* Hero section with call-to-action
* Embedded YouTube video
* Statistics section (students, trainers, etc.)
* Courses showcase cards
* Student reviews section
* Map and campus information section
* Footer with multiple columns
* Google Fonts integration (Roboto)
