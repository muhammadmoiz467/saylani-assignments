# 🏗️ Bootstrap Assignment 1 - MG Architect Website

## 📌 Project Description

Yeh ek modern responsive website hai jo **HTML, CSS aur Bootstrap** use karke banayi gayi hai. Yeh project **Saylani Mass IT Training Program (SMIT)** ke assignment ka hissa hai, jisme Bootstrap framework ka use karke layout aur components design kiye gaye hain.

## 🚀 Live Preview

👉 https://moiz-bootstrap1.surge.sh

## 🏫 Program

* Saylani Mass IT Training Program (SMIT)

## ✨ Features

* Fully responsive layout (Bootstrap Grid System)
* Navigation bar with dropdown menu
* Hero section with overlay text
* Project gallery section
* About section with team cards
* Contact form
* Map section
* Sticky header (navbar)
* Clean and modern UI
